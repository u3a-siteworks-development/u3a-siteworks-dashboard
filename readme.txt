Customise the dashboard presented to logged in users in the role of author or contributor.

== Description ==

This plugin is part of the SiteWorks project.  
It adds a SiteWorks information panel to the dashboard for all users.
It provides a customisable dashboard panel for logged in users with the role of Author or Contributor.
The administrator can also enable the customised dashboard for users with the role of Editor.

For more information please refer to the [SiteWorks website](https://siteworks.u3a.org.uk/)

== Frequently Asked Questions ==

Please refer to the documentation on the [SiteWorks website](https://siteworks.u3a.org.uk/u3a-siteworks-training/)
